# TBGAT

Package Repository

## For Developer
**Create a venv**
```powershell
python -m venv .venv
```
**install poetry**
```powershell
pip install poetry
```

**install packages**
Install the project
```powershell
poetry install
```

Install optional dev dependencies as well
```powershell
poetry install --with dev
```

Install with extras
```powershell
poetry install --extras ner
```
