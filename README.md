# TBGAT

Package Repository

## For Developer
**Create a venv**
```powershell
python -m venv .venv
```
**install poetry**
```powershell
pip install poetry
```

**install packages**
Install the project
```powershell
poetry install
```

Install optional dev dependencies as well
```powershell
poetry install --with dev
```

Install optional ner dependencies as well
```powershell
poetry install --with ner
```

You can also install everything at once
```powershell
poetry install --with dev,ner
```
